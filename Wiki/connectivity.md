# Connectivity and Setup

## Overview
Ragnarock supports both wired and wireless input paths. The system is designed for simple setup, predictable behavior, and stable performance across all supported input modes.

---

# Input Options

## Analog Input
The analog input path provides the lowest latency:
- Latency: 11 ms
- Suitable for gaming, video playback, and real-time monitoring

The signal is digitized by the HiFiBerry ADC and processed at 24-bit, 384 kHz resolution.

## Bluetooth
Bluetooth provides convenient wireless connectivity:
- Latency: 112 ms
- Suitable for casual listening
- Not recommended for real-time applications

## AirPlay
AirPlay offers high-quality streaming with significant buffering:
- Latency: 2011 ms
- Suitable for music playback
- Not suitable for synchronized audio/video use

---

# Setup Process

## Power
The system is powered by a 20 V, 5 A supply. No additional configuration is required.

## Input Selection
Ragnarock automatically prioritizes active inputs. Switching between inputs is seamless and does not require user intervention.

## Placement
While no calibration is required, placement affects low-frequency behavior. The 31 Hz tuning interacts with room boundaries, and users may adjust placement to achieve the desired bass response.

---

# Troubleshooting

## Wireless Dropouts
- Ensure line-of-sight between source and speaker
- Reduce interference from other 2.4 GHz devices

## Input Conflicts
- Disconnect unused wireless sources
- Cycle power to reset input priority

## Latency Issues
- Use analog input for real-time applications
- Avoid AirPlay for synchronized audio/video playback

---

# Design Intent
The connectivity system is designed to provide:
- Reliable wireless performance
- Low-latency wired operation
- Simple setup
- Predictable behavior across devices
